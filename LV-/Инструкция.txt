Чтобы всё работало надо:
1. Запустить файл readBot.py (это файл запуска)
2. Введи боту команду /start
* Сайт находится по id или ссылке домена для которого он сделан, если нет домена, то посмотреть сайт можно по ссылке http://localhost:5000/
* Если домена нет, то запустить сайт с другого устроиства не получится
